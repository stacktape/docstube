# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via [GitHub's private security advisories](../../security/advisories/new), or email **security@stacktape.com**. We'll acknowledge your report as quickly as we can and keep you posted on a fix.

## Scope

docstube is pre-release and the codebase is still taking shape. Even so, we take security seriously - especially anything touching credentials, tokens, or generated output, since docstube can read a user's code and drive their app. Responsible disclosure is appreciated.
