## Hero

**H1 —** AI-native docs. Verified. Always current. Yours.

**Sub —** docstube's agents read your codebase and write the whole docs site. Every page is fact-checked against the source before it ships. You supervise where you want to.

**Primary CTA —** `npx docstube wizard`

**Secondary CTA —** Read how the gate works

**Hero visual —** A single finished docs page on a clean background, one green check in the corner: "passed the gate." Nothing else. The whitespace is the point.

## What it does

Point it at your repo. Answer a few questions — doc type, who's reading, a little context. The agents write, fact-check, and ship. When your code changes, only the affected pages regenerate.

Open source. Runs on the AI you already pay for.

## Three proofs

- **Verified.** If a code sample doesn't compile, the page doesn't ship. Imports resolve against the real package; the agent that writes isn't the agent that judges. Autonomy is safe because nothing clears the gate unchecked.
- **Always current.** Change a function and only the dependent pages regenerate. The GitHub Action opens a PR with the diff — it never pushes silently.
- **Yours, on your AI.** A self-contained Astro + React site you own, MIT, zero lock-in. Runs on your Claude, Codex, or Gemini — or any compatible API — capped at the spend you set.
